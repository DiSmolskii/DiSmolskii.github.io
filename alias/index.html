Alias - Best Party Game

IOS & Android Application
01. Project Information.

About Project.

Overview

Alias is one of the best games for friends worldwide, where you need to explain words using other words, synonyms, or opposites! The team that correctly guesses as many words as possible in all rounds wins the game!

Challenge

Our goal is to create a mobile version of the original board game Alias. The game preserves all the fundamental elements and rules of the original game, but new features and capabilities have been added. We have also developed a diverse set of dictionaries for comfortable gameplay.

07. ASO Resources

Design in AppStore & GooglePlay.

We have prepared visual resources such as screenshots, icon design, promo videos, and other promotional materials. As thousands of apps are released daily, standing out from the crowd is important. In doing so, we still remember App Store recommendations.

Furthermore, to drive more visitor traffic, we conducted A/B Testing. By comparing the test results, we were able to determine which variant would increase the number of installations of our application.
